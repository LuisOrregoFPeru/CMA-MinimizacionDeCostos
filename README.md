# CMA • Minimización de Costos

Aplicación Streamlit de un solo módulo.

## Uso local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy en Streamlit Community Cloud
1. Crea un repo en GitHub y sube estos archivos.
2. En https://share.streamlit.io selecciona el repo y pon `app.py` como **Main file**.
3. Deploy.
